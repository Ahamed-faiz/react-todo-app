# React Todo List App

A dynamic task management application built with React.js as part of the Naan Mudhalvan 2.0 program.

## Features
- Add and delete tasks in real time
- Component-based architecture using React
- Clean and responsive UI

## Tech Stack
- React.js
- JavaScript
- HTML5 & CSS3

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
