# React Todo List App

A dynamic task management application built with React.js featuring 
persistent storage and a clean, responsive UI.

## Features
- Add and delete tasks in real time
- Mark tasks complete with strikethrough toggle
- Tasks persist after page refresh using localStorage
- Clean responsive UI with modern styling

## Tech Stack
- React.js (Hooks - useState, useEffect)
- JavaScript
- HTML5 & CSS3
- localStorage API

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
